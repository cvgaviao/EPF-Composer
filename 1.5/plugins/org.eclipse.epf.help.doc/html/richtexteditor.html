Rich Text Editor

The Rich Text Editor provides simple text formatting functions that allow an author to change how sections of text appear in a published Web site. The Rich Text Editor allows individual components of text to be changed within the context of the overall style sheet(s) that determine the style and format of a published Web site.

The Rich Text Editor allows you to:
  • Copy, cut and paste text. Simply highlight the text you want to cut or copy and then click the Cut or Copy buttons. To paste text, place the cursor where you want to include the text and then click the Paste button .
  • Remove all the contents in the text editor window at once by clicking the Clear Content button .
  • Change the appearance of your text by making it bold, italicized, or underlining it. Highlight the text you want to change and then click the Bold , Italics , or Underline button.
  • Create Super or Sub script by clicking the Superscript or Subscript buttons.
  • Add numbered or bulleted lists using the ordered or unordered list buttons.
  • Indent or Outdent text using the Indent or Outdent buttons.
  • Find and replace text. Click the Find/Replace button . The Find/Replace dialog box appears. Type the text you want to find in the Find field and, if you are replacing that text, type the replacement text in the Replace with field. You can search forward or back and specify whether you want your search to be case sensitive or search for only parts of words.
  • Add or create hyperlinks using the Add link button . For more information see Add a Reference or Hyperlink.
  • Add images. Click the Add Image button . The Add Image dialog box appears. Browse for the image you'd like to add and then click OK.
  • Add tables. Click the Add Table button . The Add Table dialog box appears. Specify the table parameters and then click OK.
