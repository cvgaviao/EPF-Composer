Configuration View

The Configuration view shows the content elements in a library filtered by a configuration.

A configuration is a subset of the method library content. Once a configuration is selected in the Configuration selection box, the Configuration view is refreshed with the content from the selected configuration. The content in the Configuration view is always presented in the same way in standard categories, processes, custom categories, and guidance. The Configuration view does not show any physical folders. Folders are only shown in the Library view.
