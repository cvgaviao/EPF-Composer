Perspectives

A Perspective defines a set of views and functions available for specific tasks. Eclipse Process Framework Composer contains two working perspectives: an Authoring Perspective and a Browsing Perspective.

The Authoring Perspective provides views and functions to navigate and author method content and processes, while the Browsing Perspective, as its name implies, allows you to preview and navigate through a Method Configuration without making any changes.

To select a perspective, click the Open Perspective button .
