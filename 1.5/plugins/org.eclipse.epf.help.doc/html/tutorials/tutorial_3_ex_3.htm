Exercise 3: Adding base method content associated with a task using Contribution

The purpose of this exercise is to extend a Task in RUP using ‘contribution'.

  1. Switch to the Authoring perspective.

  2. Within the Contribution Test package create a new Task: my_contributing_design_user_interface.

  3. Add a Brief Description, Main description , Purpose, etc.,: Contributing Brief Description text, Contributing Main Description text, and Contributing Purpose text.

  4. Add two new steps: Contributing Step 1 and Contributing Step 2.

  5. In Variability Relationship select Contributes, and for base task select, design_user_interface.

  6. Go back to the Steps tab and click the Order button. You can now insert your new steps into the original steps.

  7. Save the new task by closing the task editor.

  8. Switch to the Browsing perspective and view the Design the User Interface task in the Disciplines folder in Configuration View . You will find it under the Analysis_& Design discipline.

  9. Look at the descriptions and the steps to see how your new material was contributed. You should see the text that you included in the Contributing role displayed in the User Interface task.

  10. You can also try specifying additional artifacts , guidance , additional performer roles, etc., in the contributing task (in Authoring perspective) and then view the result in the Configuration View to see the behavior.
