Exercise 1: Browse Process Content using Browsing Perspective

The purpose of this exercise is to get a basic understanding of processes in EPF Composer.

(Note that this is a repeat of exercise 2 in the Browse the Library tutorial.)

  1. Make sure you are in the Browsing perspective.

  2. Select the My RUP for Small Projects configuration. If you have not created this configuration, select the RUP for Small Projects configuration.

  3. To view a Capability Pattern, click Processes > Capability Patterns > Classic RUP > Discipline Workflows > Analysis and Design. A preview of the published Web page for the selected Capability Pattern is displayed.

  4. A Capability Pattern contains a large amount of information and is displayed over four tabs – Description, Work Breakdown Structure, Work Product Usage, and Team Allocation. Explore the information on each tab.

  5. To view a Delivery Process, click Processes > Delivery Processes > Classic RUP Lifecycle. The display for a Delivery Process is similar to a Capability Pattern.
