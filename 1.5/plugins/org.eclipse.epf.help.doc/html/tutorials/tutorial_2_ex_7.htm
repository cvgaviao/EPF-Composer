Exercise 7: Manage Guidance

Guidance can be attached to any Core Method Content or Process Element. Guidance can even be attached to a Guidance element.

The goal of this exercise is to carry out some simple guidance management tasks (i.e., adding and removing guidance.)

  1. Navigate to the artifact my_rup_navmap that you created in exercise 2 in this tutorial.

  2. Add the Concept Paper process_view from the RUP plug-in.

    3. The list of items in the Select Dialog is restricted to just those items that are allowed to be added. The list can be long though. You can enter a search string, such as ‘pro' in the ‘Name patterns' box to filter the items that are displayed. Note that you can use the Collapse All and Expand All buttons on this dialog to change the way the tree is displayed.

  3. Add my_create_physical_appearance_checklist Checklist.

  4. Click the Preview tab to view your work.

  5. Remove the Concept Paper process_view.

  6. Click the Preview tab to view your work.
