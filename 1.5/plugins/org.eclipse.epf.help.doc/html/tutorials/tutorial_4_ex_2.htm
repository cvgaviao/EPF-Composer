Exercise 2: Create New Custom Category

The purpose of this exercise is to understand and use Custom Categories .

Custom Categories allow you to categorize content according to any scheme you want. Custom Categories can then be used to compose publishable Views providing a means to organize method content prior to publishing.

  1. Make sure you are in the Authoring perspective.

  2. Navigate to My Plug-in > Method Content > Custom Categories. If you have not already created this plugin, create it now, or go through the Create Core Method Content tutorial.

  3. Create a new custom category named, my_custom_category. Right-click on the Custom Categories folder, highlight New, and then click Custom Category .

  4. Open the Assign tab. If you have previously created my_content_package, add the roles , artifacts, tasks, and guidance from it into the custom category. If you do not have this package, add content from another area in the library.

    5. Note: You can do multiple select using shift or control key .

  5. Add the rup_analysis_design_discipline. You will find this under RUP > Disciplines > rup_disciplines.

  6. Open one of the elements you assigned to the new custom category. Observe how the custom category is now reflected in the Categories tab.
