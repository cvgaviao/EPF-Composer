Tutorial: Navigate the User Interface

Introduction

The purpose of this tutorial is to get familiar with user interface and the way method and process content is stored and viewed.

A Method Library consists of a series of method plug-ins. Within a plugin, content is separated into Method Content (tasks, roles, work products and guidance) and Processes (capability patterns and delivery processes). This is how the Library View, which shows all the content in the library is organized.

A configuration is a subset of the method library content. The Configuration View shows the content in the currently selected configuration. Once a configuration is selected, the configuration view is refreshed with the content from the configuration. The content in the configuration view is always laid out using the same set of folders regardless of which configuration is selected.

EPF Composer has two ‘perspectives'. The Authoring Perspective provides full browsing and authoring capabilities. The Browsing Perspective allows browsing content in the currently selected configuration.

Content is created and maintained using a series of editors.
