Exercise 2: Browse Method Content via Browsing Perspective

The purpose of this exercise is to become familiar with browsing method content in a library using the browsing perspective.

  1. Select RUP for Small Projects in the current configuration drop-down list located underneath the main menu bar.

  2. Select Browsing perspective. The Configuration View appears on the left.

  3. To view a Task, click Disciplines > RUP Disciplines > Analysis & Design > Design the User Interface. A preview of the published Web page for the selected element is displayed.

  4. Click the Primary Performer link in the relationships section. This will show the preview page for the User-Interface Designer.

  5. You can click on any links in the preview and use the back icon in the preview toolbar to return to previous pages.

  6. View other elements in the Configuration View tree – e.g. under Domains, Role Sets, etc.
