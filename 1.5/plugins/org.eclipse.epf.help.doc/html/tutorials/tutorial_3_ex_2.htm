Exercise 2: Adding to base method content associated with a work product using Contribution

The purpose of this exercise is to extend a Work Product (Artifact) in RUP using ‘contribution'.

  1. Switch to the Authoring perspective.

  2. Within the Contribution Test package create a new artifact called my_contributing_navmap.

  3. Add a Brief Description, Main Description, Purpose, etc., : Contributing Brief Description, Contributing Main Description, and Contributing Purpose.

  4. In Variability Relationship select Contributes , and for base artifact select rup_navmap. Save by closing the Work Product editor.

  5. Switch to the Browsing perspective and view the Navigation Map artifact in the Analysis and Design Domain folder in Configuration View.

  6. You should see the text that you included in the Contributing role displayed in the User-Interface Designer.
