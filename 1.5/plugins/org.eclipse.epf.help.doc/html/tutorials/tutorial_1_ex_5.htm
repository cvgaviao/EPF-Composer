Exercise 5: Search for Method Content

The purpose of this exercise is to become familiar with the search function in Eclipse Process Framework Composer.

  1. Switch to Authoring perspective.

  2. Open up the Search dialog by :

    3. •  Clicking on the Flashlight icon in the toolbar

    Or

    •  Selecting Search / Search in the main menu.

  3. Click on the Method Search tab. Enter Database in the text box and click on Search. It will take a few seconds to find all the hits.

  4. Drill down in the search results and double-click on an element. The editor for the selected item is opened. If you have Link with Editor selected in the library view, the library view display changes to show you where the selected item is in the library.

  5. Open the search dialog again and repeat the same search but this time reduce the scope to just look for tasks. Uncheck all the other content types in the scope panel. The search should be quicker and result in fewer hits.

  6. Open the search dialog again and repeat the same search but this time add rup_* as a file name pattern. Make sure the scope is still set to tasks. The search should be even quicker and result in even fewer hits.

  7. Experiment with other Method Searches.
