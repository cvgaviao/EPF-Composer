Exercise 6: Create Guidance

The purpose of this exercise is to create method Guidance elements. There is a range of different types of guidance that can be created. For this task we will simply create a representative guidance element, a checklist. Once you have done this, try authoring additional Guidance types.

In a checklist, Check Items are created in the same way as Steps in a task are created using the Check Items tab in the Checklist Editor.

  1. Make sure you are in the Authoring perspective.

  2. Within My Content Package right-click on the Guidance folder.

  3. Highlight New and then click Checklist.

  4. Name the guidance, my_create_physical_appearance_checklist, and provide the presentation name, My Create Physical Appearance Checklist.

  5. Add the following check items:

    6. •  Create branding specifications

    •  Define physical specifications

  6. Add the checklist to the my_prototype_user_interface task and then look for it on the preview page.
